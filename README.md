## 0.32.71 TEST data-active 同组状态急救

- 以 0.32.70 TEST 为唯一基线。

## 0.32.68 TEST 源码恢复

本测试版将“一次性交互诊断”升级为“一次性兔子镜总诊断”。启动后点击异常消息，可同时检查：

- 智能交互急救路线及隐藏内容状态；
- 代码块急救与智能交互急救开关；
- 标准 pre/code 外壳数量；
- 完整纯文字 `<toto data-rabbit-mirror="true">...</toto>` 候选；
- 清洗/解析测试是否可成功；
- 原始 mes 与显示层、display_text 是否可能不一致；
- 未恢复更可能属于识别失败、触发时机、显示源冲突或后续重绘覆盖。

本版不改变现有急救判断和 Prompt，只扩展诊断能力。
