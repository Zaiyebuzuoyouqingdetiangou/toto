本版把 RabbitMirror 既有急救能力真正接入每条兔子镜后的“小小维修兔”，不再只是巡逻与菜单外壳。

维修兔按当前这一面兔子镜局部调用：
- 代码块与高亮代码壳恢复
- 完整纯文字兔子镜恢复
- 原始消息临时副本恢复
- 损坏 SVG Data URI 保主体
- CSS 注释损坏修复与作用域隔离
- ID / label for / radio name / CSS ID 引用隔离
- Touch Hover 兜底
- 已验证的 checkbox、radio、按钮、显隐层、弹层、列表详情、状态兄弟映射、data-active / class 状态程序等交互急救路线
