## 0.32.9 可逆交互＋Label 内弹层急救

- 新增统一可逆状态基线：第一次接管时保存原始样式与文字，取消交互时精确恢复。
- 修复宿主克隆或重建当前 DOM 后，监听器丢失、交互停留在打开状态的问题；新 DOM 会重新绑定，但不会把打开后的样式误记成初始状态。
- 同一个 checkbox/radio 只允许一条渲染后结构急救路线接管，避免多个兜底重复写样式。
- 新增 label 内“触发文字隔在中间＋隐藏绝对定位弹层”识别；显示时临时转为不被 overflow 裁切的流式内容，第二次点击恢复原弹层样式。
- label 后置结果层与相邻隐藏内容可点击结果区返回；遮罩揭示也支持 DOM 重建后继续开关。
- 一次性诊断继续保留，并显示当前结构型急救路线。
- Prompt、配色节奏、内部折叠冷却与 Visual Scenery 规则均未修改。

# RabbitMirror 0.31.92

本版以 0.31.91 为底座，仅增强智能交互急救。
