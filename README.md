# RabbitMirror 0.33.6 TEST｜小小维修兔 v1.5

本版以 0.33.5 TEST 为唯一代码基线。

## 小小维修兔用户维修菜单

点击每条兔子镜标题后的维修兔，会打开只针对当前这一面兔子镜的本地菜单：

- 🔍 先让维修兔巡逻
- 🖱️ 点了没有反应
- 📄 显示了一堆代码
- 🎨 样子不对
- 🔧 全部试试（强制维修）
- 📋 生成全链路诊断

即使维修兔自动判为绿灯，用户仍可直接指定问题并维修，不再受自动判断限制。

所有菜单和操作均为插件本地 UI，不发送聊天消息，不增加模型 token。

## 安全边界

- 一只维修兔只处理当前一面兔子镜。
- 普通巡逻仍遵循“没有证据就不修改”。
- 强制维修只有用户明确点击后才执行。
- 不写回聊天原文、swipe 或 display_text；源码恢复仍使用临时显示副本。
- 无法确认或修复时可生成当前条目的全链路诊断。

## 保留功能

- 101 条主题元素、177 条展现形式。
- Menu QR v2.1。
- 代码块／纯文字恢复、交互恢复、SVG Data URI 保主体、CSS/ID 作用域恢复。
- RabbitMirror 全链路诊断、思维链隔离、可读性规则和暗色冷却。
