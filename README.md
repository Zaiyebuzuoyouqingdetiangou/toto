## 0.32.74 TEST｜Menu QR v1.1 自定义点菜

本版以 0.32.73 TEST 为唯一基线。

### 新增

- 点菜 QR 改为独立版本：`RabbitMirror Menu QR v1.1`。
- 插件本体版本与 QR 版本分开管理；插件更新不代表 QR 必须升级。
- 点菜主菜单新增“✍️ 自定义点菜”：无需浏览分类树，直接输入想看的主题、媒介、氛围或其他要求。
- 自定义点菜会生成一次性订单：等待下一次实际生成的兔子镜，采用一次后失效；不影响正文，不延续到后续兔子镜。
- 下载按钮显示 QR 版本，并下载 `assets/RabbitMirror-MenuQR-v1.1.json`。
- 提示文字保持为：“下载后请在快捷回复中手动导入。”

### 保留

- 101 条主题元素与 177 条展现形式的三级浏览点菜树。
- SVG Data URI 保主体急救与结构截断诊断。
- `data-active` 同组状态急救、源码恢复、思维链隔离、暗色冷却及全部既有急救路线。
- Prompt 与母本内容未修改。

### Token

QR JSON 作为插件静态资源存在，仅在点击下载按钮时由浏览器下载，不注入 Prompt、聊天记录或模型上下文，不增加聊天前及每轮 token。
