# RabbitMirror 0.33.44 TEST｜正文流安全＋免费成图网站输入修复

## 0.33.44 修复

- 恢复“免费成图网站”输入框，免费成图模式可由用户填写地址模板。
- 自定义 API 模式继续支持接口地址、API Key、连接并拉取模型与模型选择。
- 两种文生图模式继续统一使用 2D 动漫画风；关闭时不构建图片 Prompt、不调用接口。
- 修正主 Prompt 内部冲突：主要正文、长句、段落与关键反馈必须进入正常文档流；position:absolute、transform、自由拼贴与叠层只能用于装饰、短标签和辅助信息。
- 保留复杂交互、媒介本体优先和视觉完成度要求，不再允许用固定画框、绝对定位长正文或 overflow 裁切来满足复杂度。
- 维修兔仍保持用户点击后才执行；本次正文溢出修复发生在生成 Prompt，不依赖维修兔。
