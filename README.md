# 兔子洞小剧场 / Rabbit Hole Theater

Toto v0.21.0

## v0.21.0 更新

- 新增抽取模式：`仅展现形式` / `主题元素 + 展现形式（经典模式）`。
- 审核规则随抽取模式变化：仅展现形式模式不抽主题元素，主题元素栏固定为“未抽取”。
- Visual Scenery 规则独立为 `visualSceneryRules.js`，仅 Visual Scenery 轮次注入，并加入“自由取景”规则，减少机械复刻当前场景。
- 执行边界改为“不得干预或改写主线叙事”，不再用“引用/暗示”压制正文氛围与意象。
- 中文锁定继续独立为 `runtimeLanguageRules.js`，塔罗图片编号规则继续仅塔罗相关轮次注入。
- 保留 10 轮冷却：主题、展现形式、UI美化构思，以及近期 UI美化构思记录。
- 保留轻量运行版规则，避免回到每轮大段常驻注入。

## 安装

将本仓库放入 SillyTavern 的第三方扩展目录，或通过 Extensions 安装 GitHub 仓库地址。

## 说明

完整主题库与展现形式库仍保存在 `data/raw/` 与 `data/structured/` 中。插件每轮只注入运行必要规则、本轮抽到的主题/展现形式片段、近期 UI美化构思冷却记录，以及必要的条件规则。
