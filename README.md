# 兔子洞小剧场 / Rabbit Hole Theater

Toto v0.19.3

这是 SillyTavern 第三方扩展。它会在生成前自动注入小剧场协议，不需要在原预设里保存整段规则。

## v0.19.3 更新

- 强化简体中文锁定：标题、状态栏、按钮、系统提示、UI 标签等可见文字也必须中文化，禁止英文承担主要界面标签。
- 冷却机制扩大到 10 轮：主题、展现形式、UI美化构思都会尽量避开最近 10 轮重复。
- 增强丰富版式抽取：随机时更偏向带界面结构、视觉锚点和 UI 层级的展现形式，减少纯文字类连续出现。
- 增加 UI美化构思冷却：每轮注入整体视觉方案，并将近期 UI美化构思作为禁用项，降低“同一面板换字”的概率。
- 强化小剧场启动：将小剧场作为本轮回复格式的固定组成部分。
- Visual Scenery 模式继续使用视觉画布优先协议。

## 安装

将仓库根目录保持为：

```txt
manifest.json
index.js
style.css
README.md
data/
src/
```

在 SillyTavern 的 Extensions / Install Extension 中粘贴仓库地址安装。

## 设置

- 启用兔子洞自动注入
- 输出 `<thinking>` 执行摘要
- 正文指令优先
- Visual Scenery 动态渐变模式
- UI 自查优化 / 丰富版式
- 10轮冷却：避免重复主题/展现形式/UI美化构思
- 跳过 quiet 后台生成
- 跳过 impersonate 生成
- 控制台调试日志

Toto 仅显示在插件设置界面，不会生成到小剧场内部。
